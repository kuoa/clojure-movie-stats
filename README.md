# Movie Data Science

This clojure learning project is about extracting and processing data from the [MovieLens](http://movielens.org) dataset.

### Compilation

```sh
Compiling with lein
lein run

Compiling into a .jar
lein uberjar
java -jar movie-stats-1.0.0-SNAPSHOT-standalone.jar
```

### Requirements

* Java >= 6
* Clojure 1.5
* Leiningen
